# Student Management System using Linked List in C

Mini Project for Data Structures using C Programming

## Introduction

This project is a Student Management System developed in C using Singly Linked List Data Structure.  
The system stores student records dynamically and performs various operations such as insertion, deletion, searching, updating, sorting, and percentage calculation.

The project demonstrates the practical implementation of linked lists, structures, pointers, dynamic memory allocation, and traversal operations in Data Structures.

---

## Features

- Add New Student Record
- Display Student Records
- Search Student by Roll Number
- Update Student Details
- Delete Student Record
- Count Total Students
- Find Student with Highest Percentage
- Display Students Above 75%
- Pass and Fail Student List
- Sort Students by Percentage
- Insert Student at End
- Delete Student at End
